HOW TO USE

Each page is a single self-contained HTML file. Pick one, save it as 404.html in your site root, and your host will serve it for missing routes.

Cloudflare Pages, Netlify, GitHub Pages: drop 404.html in the root of your published directory.

Nginx / Apache: point error_page 404 / ErrorDocument 404 at the file.

# nginx
error_page 404 /404.html;

Want to mix two pages on one site (e.g. one path uses the BSOD, another uses the Society)? See the original goolz 404.html — it switches by location.pathname.
